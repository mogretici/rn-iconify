# Animations

Add eye-catching animations to your icons with 6 built-in presets.

## Built-in Presets

```tsx copy
import { Mdi } from 'rn-iconify';

<Mdi name="loading" animation="spin" />      // Continuous rotation
<Mdi name="heart" animation="pulse" />       // Scale up/down
<Mdi name="arrow-down" animation="bounce" /> // Bouncing effect
<Mdi name="bell" animation="shake" />        // Side-to-side shake
<Mdi name="circle" animation="ping" />       // Expanding ping
<Mdi name="hand-wave" animation="wiggle" />  // Wiggle rotation
```

## Animation Comparison

| Preset | Effect | Use Case |
|--------|--------|----------|
| `spin` | 360° rotation | Loading indicators |
| `pulse` | Scale 1 → 1.2 → 1 | Attention, heartbeat |
| `bounce` | Vertical bounce | Download, scroll hint |
| `shake` | Horizontal shake | Error, notification |
| `ping` | Expand + fade | Status, notification |
| `wiggle` | Rotate ±15° | Greeting, attention |

## Loading Spinner

Create a loading indicator:

```tsx copy
import { Mdi } from 'rn-iconify';
import { View, Text } from 'react-native';

function LoadingState() {
  return (
    <View style={{ flexDirection: 'row', alignItems: 'center', gap: 8 }}>
      <Mdi name="loading" animation="spin" size={24} color="#6366f1" />
      <Text>Loading...</Text>
    </View>
  );
}
```

## Notification Bell


## Like Button

Animated heart icon:

```tsx copy
import { useState } from 'react';
import { TouchableOpacity } from 'react-native';
import { Mdi } from 'rn-iconify';

function LikeButton() {
  const [liked, setLiked] = useState(false);
  const [animating, setAnimating] = useState(false);

  const handlePress = () => {
    setLiked(!liked);
    setAnimating(true);
    setTimeout(() => setAnimating(false), 600);
  };

  return (
    <TouchableOpacity onPress={handlePress}>
      <Mdi
        name={liked ? 'heart' : 'heart-outline'}
        size={32}
        color={liked ? 'red' : 'gray'}
        animation={animating ? 'pulse' : undefined}
      />
    </TouchableOpacity>
  );
}
```

## Conditional Animation

Animate based on state:

```tsx copy
import { Mdi } from 'rn-iconify';

interface IconProps {
  name: string;
  isLoading?: boolean;
  hasError?: boolean;
  isNew?: boolean;
}

function StatusIcon({ name, isLoading, hasError, isNew }: IconProps) {
  let animation: 'spin' | 'shake' | 'pulse' | undefined;

  if (isLoading) animation = 'spin';
  else if (hasError) animation = 'shake';
  else if (isNew) animation = 'pulse';

  return <Mdi name={name} animation={animation} />;
}
```

## Animation Duration

Control animation speed with iteration count (animations loop by default):

```tsx copy
// Continuous animation
<Mdi name="loading" animation="spin" />

// Animation runs based on preset duration
<Mdi name="bell" animation="shake" />
```

## With React Navigation

Animated tab icons:

```tsx copy
import { createBottomTabNavigator } from '@react-navigation/bottom-tabs';
import { Mdi } from 'rn-iconify';

const Tab = createBottomTabNavigator();

<Tab.Navigator>
  <Tab.Screen
    name="Notifications"
    component={NotificationsScreen}
    options={{
      tabBarIcon: ({ focused, color, size }) => (
        <Mdi
          name="bell"
          size={size}
          color={color}
          animation={focused ? 'pulse' : undefined}
        />
      ),
    }}
  />
</Tab.Navigator>
```

## Tips

### 1. Use Sparingly

Too many animations can be distracting:

```tsx
// ✅ Good - one animated icon
<View>
  <Mdi name="loading" animation="spin" />
  <Mdi name="check" />
  <Mdi name="close" />
</View>

// ❌ Avoid - too many animations
<View>
  <Mdi name="loading" animation="spin" />
  <Mdi name="heart" animation="pulse" />
  <Mdi name="bell" animation="shake" />
</View>
```

### 2. Meaningful Animations

Use animations that convey meaning:

```tsx
// ✅ Good - loading uses spin
<Mdi name="loading" animation="spin" />

// ✅ Good - error uses shake
<Mdi name="alert" animation="shake" />

// ❌ Confusing - loading with bounce
<Mdi name="loading" animation="bounce" />
```

### 3. Respect Motion Preferences

Check user's reduced motion preference:

```tsx copy
import { AccessibilityInfo } from 'react-native';
import { useEffect, useState } from 'react';
import { Mdi } from 'rn-iconify';

function AccessibleIcon() {
  const [reduceMotion, setReduceMotion] = useState(false);

  useEffect(() => {
    AccessibilityInfo.isReduceMotionEnabled().then(setReduceMotion);
  }, []);

  return (
    <Mdi
      name="loading"
      animation={reduceMotion ? undefined : 'spin'}
    />
  );
}
```
