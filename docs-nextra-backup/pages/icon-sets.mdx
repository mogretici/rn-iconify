## Popular Icon Sets

### Material Design Icons (Mdi)

The most comprehensive icon set with 7,447+ icons covering almost every use case.

```tsx copy
import { Mdi } from 'rn-iconify';

<Mdi name="home" />
<Mdi name="settings" />
<Mdi name="account" />
```

**Best for**: General purpose apps, Android-style design

---

### Heroicons

Beautiful hand-crafted icons by the makers of Tailwind CSS. Available in outline and solid styles.

```tsx copy
import { Heroicons } from 'rn-iconify';

<Heroicons name="home" />
<Heroicons name="user" />
<Heroicons name="cog-6-tooth" />
```

**Best for**: Modern web-style apps, Tailwind users

---

### Lucide

## Choosing an Icon Set

| Need | Recommended Set |
|------|-----------------|
| Most icons | Material Design (Mdi) |
| Modern style | Heroicons, Lucide |
| Multiple weights | Phosphor (Ph) |
| Brand logos | Font Awesome Brands |
| iOS style | SF Symbols, Ionicons |
| Android style | Material Design |
| Dashboard | Tabler, Carbon |

## Using Multiple Sets

You can use icons from different sets in the same app:

```tsx copy
import { Mdi, Heroicons, Lucide } from 'rn-iconify';

function MyComponent() {
  return (
    <>
      <Mdi name="home" />
      <Heroicons name="user" />
      <Lucide name="camera" />
    </>
  );
}
```

## Import Pattern

All icon sets follow the same import pattern:

```tsx
// Component import
import { ComponentName } from 'rn-iconify';

// Type import
import { type ComponentNameIconName } from 'rn-iconify';
```

Examples:
- `Mdi` → `MdiIconName`
- `Heroicons` → `HeroiconsIconName`
- `Lucide` → `LucideIconName`
- `Ph` → `PhIconName`
